# 大场面高动态分镜规则

## 一、基础结构

### 时长分配（15秒黄金法则）

```
总时长：15秒
├── 序列一（5s）：蓄势/爆发起点
├── 序列二（5s）：核心动态/高潮
└── 序列三（5s）：定局/转折/收尾

每个序列：2-3个镜头
每个镜头：1.5-2.5秒
快切镜头：0.3-0.5秒/镜
```

### 镜头数量控制

| 场景类型 | 推荐镜头数 | 说明 |
|---------|-----------|------|
| 15秒短片段 | 7-9镜 | 保证每个镜头有足够信息量 |
| 30秒中段 | 12-15镜 | 可增加细节特写 |
| 60秒长段 | 20-25镜 | 允许慢节奏段落 |

## 二、景别与运镜组合

### 核心景别序列（推荐组合）

```
大特写（细节）→ 特写（表情）→ 中景（动作）→ 全景（场面）→ 大全景（氛围）
```

### 高动态运镜技法

| 技法 | 适用场景 | 视觉效果 |
|-----|---------|---------|
| **急速推镜** | 法诀结印、武器显形 | 强调细节，制造紧张感 |
| **急速拉远** | 法宝暴涨、场地展开 | 展示规模，形成视觉冲击 |
| **360°环绕** | 武器展示、人物旋转 | 全方位展示，增加仪式感 |
| **急速跟拍** | 飞行、冲刺、对冲 | 强化速度感，身临其境 |
| **穿透式推进** | 突破屏障、内部透视 | 制造层次，揭示隐藏 |
| **多角度快切** | 连续攻击、多角度打击 | 加快节奏，增强打击感 |
| **仰/俯拍拉升** | 升天、降临、俯瞰 | 强化威压，展示地位 |

### 景别切换公式

```
[特写] → [快切多角度] → [大全景定局]
细节展示 → 动态爆发 → 场面收束
```

## 三、镜头描述标准格式

### 单镜头模板

```markdown
**镜号 XX | 时长：Xs**
- **景别**：[大特写/特写/中近景/中景/全景/大全景]
- **运镜**：[技法 + 方向 + 速度]
- **画面**：[主体动作 + 环境互动 + 关键细节]
- **特效**：[光效/粒子/能量/虚影/物理反馈]
- **音效**：[主音效 + 辅音效 + 情绪音]
```

### 特效描述关键词库

**光效类**：
- 爆射、吞吐、流转、明暗起伏、放射状、螺旋状、涟漪波纹、光轨尾迹

**粒子类**：
- 光尘、星尘、液态光珠、灵气光丝、火花四溅、冰晶碎屑

**能量类**：
- 能量场、灵气涟漪、灵压波纹、冲击波、气场对冲、光柱冲天

**虚影类**：
- 龙形虚影、法相显现、元神外放、残影重重、分身化影

**物理反馈**：
- 空气撕裂、空间扭曲、地面龟裂、冰霜蔓延、衣袂翻飞

## 四、音效设计原则

### 层次结构

```
第一层：动作音效（打击、破空、撞击）
第二层：法宝音效（剑鸣、龙吟、绫缎飘动）
第三层：环境音效（风声、雷鸣、空间共鸣）
第四层：情绪音效（心跳、呼吸、惊呼声）
```

### 节奏控制

| 场景阶段 | 音效特点 | 示例 |
|---------|---------|------|
| 蓄势 | 低频、单一、渐强 | 「嗡——」共鸣声 |
| 爆发 | 高频、复合、炸裂 | 「轰——！」多层混音 |
| 高潮 | 连续、急促、叠加 | 「咚咚咚咚」鼓点 |
| 定局 | 戛然而止、余音绕梁 | 「铮——」后归于寂静 |

### 拟声词库

**打击类**：砰、轰、匡、铛、铮、噗、嗤
**撕裂类**：嘶啦、嗖、唰、哗
**共鸣类**：嗡、呜、吟、啸
**碎裂类**：啪啦、咔咔、噼啪
**气浪类**：呼、嘶、猎猎

## 六、序列设计模板

### 模板A：法宝显形（以斩龙剑为例）

```markdown
## 镜头序列：XX显形

**镜号 01 | 时长：2.0s**
- **景别**：大特写（虚空/剑柄/关键部位）
- **运镜**：缓慢推进，聚焦凝聚点
- **画面**：虚空扭曲，光粒汇聚，核心形态初现
- **特效**：光粒碰撞融合，形态逐寸生成，符文亮起
- **音效**：「嗡——」灵气共鸣，逐渐高亢

**镜号 02 | 时长：1.5s**
- **景别**：极速大特写三连拍（0.5s/镜）
- **运镜**：炸裂式快切
- **画面**：关键部位1 → 关键部位2 → 关键部位3
- **特效**：每镜一次灵光大爆发，画面充满又收敛
- **音效**：「轰！轰！轰！」配合心跳节奏

**镜号 03 | 时长：2.5s**
- **景别**：大特写 → 仰拍大特写
- **运镜**：环绕展示 + 急速仰拉
- **画面**：法宝完全显形，光芒万丈，剑指苍穹
- **特效**：螺旋光轨、光环扩散、光芒辐射、威压波纹
- **音效**：「嗷——！」一声长啸，达到最高潮
```

### 模板B：技能爆发（以缚神为例）

```markdown
## 镜头序列：XX爆发

**镜号 01 | 时长：2.0s**
- **景别**：中景 → 大特写
- **运镜**：急速下降跟拍 + 环绕拉升
- **画面**：施术者升空结印，法诀成型
- **特效**：法印旋转扩大，符文环绕，金光炸裂
- **音效**：法诀喝声「缚神！」，「铮——」共鸣

**镜号 02 | 时长：2.0s**
- **景别**：特写 → 多角度快切 → 大全景
- **运镜**：极速后拉 + 五角度快切(0.4s/镜) + 旋转俯拍
- **画面**：法宝暴涨，见风就长，填满空间
- **特效**：疯长过程、光丝蔓延、红绿对抗、空间填满
- **音效**：「叮——」后「轰隆隆——」万马奔腾

**镜号 03 | 时长：2.0s**
- **景别**：俯冲全景 → 环绕快切 → 微观特写
- **运镜**：急速下坠 + 360°环绕 + 急速拉远
- **画面**：分化万千，交织围困，封印定局
- **特效**：千万流星、金铁交鸣、符文流转、封印生效
- **音效**：「嗖嗖嗖——」汇聚 → 「轰——！」合拢 → 「嗡嗡——」封印
```

### 模板C：优势逆转（以怒龙破绫为例）

```markdown
## 镜头序列：优势逆转

**镜号 01 | 时长：2.0s**
- **景别**：大特写 → 仰拍中景
- **运镜**：微距慢推 + 急速拉升
- **画面**：优势方出现破绽，劣势方眼神变化
- **特效**：裂痕出现、青光渗出、表情凝固、瞳孔倒映
- **音效**：「嘶啦——」撕裂声，「什么？！」惊呼

**镜号 02 | 时长：2.0s**
- **景别**：全景 → 中景 → 正面对冲
- **运镜**：急速后退 + 侧面跟拍 + 正面推进
- **画面**：劣势方破困而出，人剑合一，势如破竹
- **特效**：茧房爆裂、龙形虚影、尾迹流光、气势暴涨
- **音效**：「嗷——！」龙啸，「嗖——轰！」音爆

**镜号 03 | 时长：2.0s**
- **景别**：特写 → 侧面快切 → 对冲大特写
- **运镜**：急速后拉 + 跟随快切 + 微晃推进
- **画面**：优势转防守，层层败退，最终绝境
- **特效**：屏障堆叠、层层碎裂、剑尖逼近、火花四溅
- **音效**：「嗤嗤嗤——」连续刺破，「铮——！」静止
```

## 七、实用技巧清单

### ✓ 画面张力增强
- [ ] 使用极端角度（超低仰拍/超高俯拍）
- [ ] 加入前景遮挡物增加层次
- [ ] 利用镜面反射（冰墙、水面、剑刃）
- [ ] 快慢节奏交替，避免匀速
- [ ] 关键帧使用黑场停顿（0.5s）

### ✓ 特效描述技巧
- [ ] 量化描述：「三尺→填满天空」「千万条」
- [ ] 动词选择：「暴涨」「疯长」「撕裂」「吞噬」
- [ ] 色彩对比：红vs绿、金vs黑、白vs血
- [ ] 物理反馈：空气扭曲、地面龟裂、冰霜蔓延

### ✓ 情绪渲染要点
- [ ] 眼神特写：瞳孔倒映、目光变化
- [ ] 微表情：嘴角上扬→凝固、眉头舒展→紧锁
- [ ] 肢体语言：从从容到慌乱、从压抑到爆发
- [ ] 呼吸节奏：急促、屏息、长叹

## 八、快速检查清单

生成完分镜后，对照检查：

```markdown
## 自检清单

- [ ] 总时长是否控制在要求范围内？
- [ ] 每个序列是否有明确的情绪转折？
- [ ] 景别是否从细节→场面→细节循环？
- [ ] 是否有至少一次多角度快切？
- [ ] 特效描述是否包含光效+粒子+物理反馈？
- [ ] 音效是否有从低到高的层次变化？
- [ ] 是否突出了主角/关键道具？
- [ ] 结尾是否有定格画面或悬念？
```

## 九、示例：完整15秒分镜框架

```markdown
**镜号 06 | 时长：1.5s** [大特写] [推进/微晃]

**镜号 08 | 时长：1.5s** [快切×4] [切换]
**镜号 09 | 时长：1.5s** [大特写→中景] [拉远/上升]

🎬 **节奏总控**：起→承→转→合
🎬 **视觉重点**：[填写核心视觉元素]
🎬 **音效高峰**：[填写最高潮音效]
```
