# AI Drama Director

AI漫剧导演 - 从剧本自动生成场景、人物和分镜

## 功能

当你给一个剧本（如 `第一章剧本.md`）时，AI导演会：

1. **分析剧本** - 提取场景和人物
2. **检查资产** - 查看 `assets/scenes/` 和 `assets/characters/` 是否已存在
3. **创建缺失** - 如果不存在则为新场景/人物生成提示词文件
4. **生成分镜** - 根据场景和人物的提示词生成分镜

注意：如果创建的文件名已经存在了，就把名字改成xxx-副本，一定不要覆盖已有文件

## 输入格式

剧本可以是任意文本格式（小说原文、剧本、故事大纲等）。AI导演会自动：
- 识别场景和地点描述
- 提取人物角色
- 分析时间、氛围等信息

支持纯文本、Markdown、剧本格式等。

## 输出结构

运行后会更新/创建：

```
assets/
├── characters/
│   ├── 人物A.md   # 人物提示词（文件名使用中文）
│   └── 人物B.md
└── scenes/
    ├── 场景A.md   # 场景提示词（文件名使用中文）
    └── 场景B.md   

scripts/
└── 第一章分镜.md   # 分镜文件（自动生成）
```

## 分镜规则

- 先从导演的角度分析剧本，然后把分析的结果从专业分解师的角度生成多组分镜
- 对齐场景和人物提示词出现的元素
- 注意每个镜头要确保对应时长可以匹配上相应的内容，避免实际内容超出分配的时长
- 每个分镜镜头一定要描写对应的镜头语言
- 关键对话不能省略，人物说话请用引号包裹
- 分镜分类：
  - 普通分镜：适用于普通镜头、文戏、日常场景，请看[references/普通分镜.md]
  - 大场面高动态分镜镜：适用于大场面、高动态、打斗场景，请看[references/大场面高动态分镜.md]

## 场景和人物的prompt

### 场景的元素

- 主题元素
- 环境与场景
- 时间与天气
- 光线与色彩
- 风格与质感：虚幻引擎5
- 构图与视角：生成多角度图片

### 人物的元素

严格遵守下面的格式：

- 性别
- 年龄
- 体型
- 脸型
- 五官风格
- 发型
- 发色
- 刘海
- 长度
- 眼睛颜色
- 瞳形
- 性格与气质
- 人种：中国人
- 画面规格：白色背景，全身正面站立姿势，适合作为角色参考图，虚幻引擎5，3D风格

## 违禁词替换参考

| 禁用词 | 替换词 |
|--------|--------|
| 鲜血 | 暗红色痕迹/能量残留 |
| 伤口 | 破损/裂痕/印记 |
| 死亡/尸体 | 静止/沉睡/失去意识 |
| 杀戮 | 制服/阻止/击倒 |
| 暴力/殴打 | 冲撞/撞击 |
| 恐怖 | 紧张/震撼 |
| 血腥 | 暗色液体/能量喷涌 |
| 残忍 | 凌厉/果决 |

## 武器物理一致性规则

| 武器类型 | 允许动作 | 打击感表现 |
|----------|----------|------------|
| 剑/刀/刃 | 砍、刺、斩、切、劈 | 切割穿透、整齐断裂 |
| 枪/矛/棍 | 穿刺、砸击、横扫 | 贯穿、震退、钝击 |
| 能量/魔法 | 释放、引爆、电弧 | 烧灼、爆炸、冲击 |
| 拳/掌/体术 | 冲撞、格挡、摔投 | 气浪、地面碎裂 |

## 注意

- 分镜文件的最末尾加上所有出现的人物和场景的清单
- 如果上一段和下一段的剧情是连续的，下一段必须复制上一段的【场景与氛围】描述，仅做微调
- 台词字幕时间点必须精确标注，与分镜时间轴对齐
